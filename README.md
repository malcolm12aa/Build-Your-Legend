⚔️ Anime RPG — YGGDRASIL Inspired Web Game

A mobile-friendly, browser-based RPG inspired by Overlord’s YGGDRASIL system, combining multiple anime universes into one customizable build system.

⸻

🌌 Overview

Anime RPG is a multi-class, multi-race RPG system where your character is built through:

* Race Selection (Anime-based origins)
* Job/Class Paths (Stackable YGGDRASIL-style system)
* Stat Growth & Skill Unlocks
* Recruitable Companions
* Turn-Based Combat

Unlike traditional RPGs, this game focuses heavily on build customization, synergy, and hidden class discovery.

⸻

⚙️ Core System — YGGDRASIL Build Mechanics

This game follows a layered progression system inspired by Overlord:

* 15 levels in a Base class or race
* 10 levels in an Advanced class
* 10 levels in a Specialist class
* 5 levels in a Rare class
* Continue mixing until Level 100

💡 A strong build is not one class to max level — it’s a synergized combination of multiple classes.
