# ⚔️ Build Your Legend — Fantasy RPG Web Game

**Build Your Legend** is a mobile-friendly browser RPG built around deep character progression, fantasy race evolution, job mastery, equipment builds, and turn-based battles.

This project started as an anime-inspired RPG system, but the current version has been rebuilt into a fantasy-style race and class progression game with branching evolution paths, job upgrades, shops, raids, challenge modes, and a growing skill system.

## 🌌 Game Overview

Build Your Legend is about creating a character and growing them through multiple long-term progression systems:

- Choose a **base fantasy race**
- Level that race to unlock **race evolutions**
- Choose a **base job/class**
- Master your current job before upgrading to the next one
- Unlock stronger skills, passives, equipment, shops, maps, bosses, and challenge content
- Build toward powerful fantasy endgame forms

The main idea is:

> **You do not become strong by picking everything at once. You become strong by mastering one path at a time.**

## 🔺 Race Evolution

Every race has evolution paths.

Race evolution is now its own separate progression track from jobs/classes.

### Race Evolution Rules

- You must max your current race before evolving it.
- Race evolution does **not** block job progression.
- Job progression does **not** block race evolution.
- Each race has unique evolution paths.
- Evolution paths can include:
  - Advanced race forms
  - Specialist race forms
  - Rare race forms
  - Hidden race forms

### Example Race Evolution Style

```text
Human → High Human → Hero Bloodline → Saint Hero → Divine Champion
Elf → High Elf → Ancient Elf → Spirit Elf → World Tree Apostle
Goblin → Hobgoblin → Goblin Knight → Goblin Lord → Goblin King
Oni → Red Oni / Blue Oni → War Oni → Oni Lord → Kishin Oni
Slime → Greater Slime → Predator Slime → Demon Slime → True Demon Slime
Demon → Greater Demon → Archdemon → Demon Duke → Demon Lord
Angel → High Angel → Archangel → Seraph → Divine Executor
Vampire → Noble Vampire → True Vampire → Vampire Lord → Progenitor
Dragonkin → Lesser Dragon → Ancient Dragon → Dragon Lord → True Dragon
```

The character status screen shows your next race evolution when your current race stage is maxed.

## 🧭 Job / Class System

The job system has also been rebuilt.

Only **base jobs** are shown in the full class registry. Stronger jobs are unlocked through progression.

### Job Upgrade Rules

- You start with one base job.
- You can only get a new job upgrade after maxing your current job.
- Job upgrades are separate from race evolution.
- A race evolution will not prevent you from upgrading your job.
- Advanced, Specialist, Rare, and Hidden jobs unlock through mastery and requirements.
